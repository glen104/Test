Rotator Column Flags
Vis — Show or hide a site from the rotation without deleting it.
Dur(s) — How many seconds to display this panel before advancing.
Ref(m) — Auto-reload interval in minutes. 0 = never auto-reload on a timer.
RoV — Refresh on View. Reloads the panel every time the rotator lands on it. Enabled by default on all radar and NWPS panels.
Label — Show the site name as an overlay at the top of the screen while it's active.
Img — Force the panel to render as a static image (like FHO/AHD) instead of a live iframe. Use this for any direct .png/.jpg URL or image endpoint that doesn't need browser interaction.
Scheduler — Task Types
Daily — Fires every day at the set time, year-round. No date range needed.
Weekday — Same as Daily but skips Saturday and Sunday.
Seasonal — Fires daily within a MM-DD to MM-DD date range. Repeats every year unless a year is hardcoded in the defaults (like the World Cup tasks).
One-Day — Fires once on a single specific date (YYYY-MM-DD). Never repeats. Perfect for one-off events, briefings, or deadlines.
HUD Buttons
▼ / ▲ OPS HUB — Collapse or restore the HUD bar.
⬡ (External Link) — Opens the stored URL for the current panel in a new tab.
⚙ (Settings) — Opens this manager.
OPEN FRAME — Attempts to capture the live URL from the current iframe (works if same-origin). Falls back to the stored URL. Opens in a new tab so you can copy the adjusted URL.
PAGE UPDATE — Opens a dialog to paste a new URL for the current panel. Warns you if the URL hasn't changed.
HARD REFRESH — Force-reloads the current panel immediately, bypassing all refresh logic.
📡 AHD Auto-Management
AHD panels are never manually managed. They are injected automatically from the NOAA product JSON and always appear immediately after the FHO in the rotation.
Detection — JSON is polled every 5 minutes. A new AHD will appear on screen within 5 minutes of hitting the web.
Expiry — Each AHD carries an expiration timestamp. The rotator checks every second and removes expired panels the moment they pass their expiry time. AHDs nominally expire on the hour but the actual timestamp from the JSON is always used.
Persistence — AHD panels are never saved to localStorage. They are rebuilt fresh from the JSON on every page load.
Tips & Gotchas
⚠ localStorage takes precedence over defaults. If you update the code with new default URLs, those won't take effect until you hit RESET DEFAULTS — which wipes saved sites, tasks, and sound settings.
⚠ Capturing adjusted radar URLs: Right-click the panel → "This Frame" → "Open Frame in New Tab". Copy the URL from that tab, then use PAGE UPDATE to save it.
⚠ Scheduled syncs (FHO auto-refresh) fire at 7:10, 14:10, and 15:10 local time.
⚠ Seasonal tasks repeat every year unless a year property is set in the defaults — like the World Cup calls.
